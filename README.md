# Fillit
Form the smallest possible square from a list of tetriminos.

## Compiling
Run `make` command and an executable called `fillit` should be created.

## Usage
`./fillit [file]`

## Examples

```
./fillit test_file

ABBBB.
ACCCEE
AFFCEE
A.FFGG
HHHDDG
.HDD.G
```
